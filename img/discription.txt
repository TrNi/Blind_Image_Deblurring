This data set included images from Dong, Levin, and Sun

1. Data description

"./data/Dong" includes 3 blurrer color images.

"./data/Levin" includes 8      kernels ("./data/Levin/kernel/"), 
                                 4      truth images ("./data/Levin/truth/"),
                                 8*4   blurred gray images ("./data/Levin/blurred/").

"./data/Sun" includes 8        kernels ("./data/Sun/kernel/"), 
                               80      truth images ("./data/Sun/truth/"),
                               8*80   blurred gray images ("./data/Sun/blurred/").

2. "*.jld" description dict{Array{Float64,2}}
For color images, each .jld file has 3 dicts: {"red", "green", "blue"}.
For gray images, each .jld file has 1 dicts: {"gray"}.
For kernel, each .jld file has 1 dicts: {"kernel"}.
